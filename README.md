# 📺 YouTube DPDP Privacy Prototype

A Product Management prototype designed as part of the **Vedantu Product Management Intern Assignment**.

This project demonstrates how YouTube's onboarding and consent flow can be redesigned to align with **India's Digital Personal Data Protection (DPDP) Act, 2023**, while maintaining a seamless user experience.

---

## 🎯 Problem Statement

Design a privacy-first onboarding experience that:

- Complies with India's DPDP Act, 2023
- Protects children's personal data
- Enables verifiable parental consent
- Improves transparency and user trust
- Maintains a smooth onboarding experience

---

## ✨ Features

- 🎬 YouTube-inspired onboarding experience
- 📅 Age verification
- 👨‍👩‍👧 Parent verification flow
- 🔐 OTP verification
- 🛡️ Privacy preferences with consent management
- 📊 Privacy Hub dashboard
- ✅ Success confirmation screen
- 📱 Responsive mobile-first UI
- 🎨 Material Design inspired interface
- 🚀 Hosted using GitHub Pages

---
